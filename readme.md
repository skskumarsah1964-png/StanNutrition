# StanNutrition Website

## Project Overview

StanNutrition is the official website for Stan Nutrition, a brand offering premium, science-backed nutritional supplements designed for fitness enthusiasts, vegans, and individuals aiming for peak performance and wellness. The site showcases products like the YSTAN Mass Gainer (Chocolate flavor), highlights key benefits, and provides easy navigation to explore plans, products, and more.

The homepage features:
- A clean, modern design with a focus on strength and nutrition.
- Prominent messaging: "Premium Nutrition for a Stronger You".
- Product spotlight on the Best Mass Gainer with full chocolate flavor.
- Call-to-action buttons like "Get Started Today".

Website: www.stannutrition.com

## Features
- Responsive layout suitable for desktop and mobile.
- Navigation menu: Home, Plans, Products, About, Contact.
- Hero section with high-quality imagery of fitness and products.
- Promotional banners emphasizing vegetarian, lab-tested, and certified products (FSSAI, GMP, ISO, HACCP).
